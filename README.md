# Project Playground

Project Playground is my sandbox project for learning Cute Framework while building a small C++20 graphics and gameplay playground.

## Build

Configure and build from the repository root:

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

Run the executable:

```bash
./build/project_playground
```
